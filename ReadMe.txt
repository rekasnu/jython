How to run

1. Open comandprompt

2. Change the directory - navigate to folder where all the files are located:

	cd 'your path'

3. compile Model.java and View.java

	javac Model.java
	javac View.java - for debugging purpose

4. Make sure that jython is added to your computers path.

5. Run the program

	jython run.py

Comment:
	
	Input: The max size for matrix should be at least 1000 in order to see some chart
